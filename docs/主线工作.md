## 字典库设计建议

字典库（Code Table）是企业管理系统中常用的功能，用于存储系统中的静态配置数据。

### 常见字典类型

| 类型 | 用途 | 示例 |
|------|------|------|
| **状态字典** | 记录状态枚举 | 在职/离职、待审批/已批准/已拒绝 |
| **分类字典** | 分类选择 | 行业分类、员工民族、职称等级 |
| **区域字典** | 地理信息 | 省份、城市、区县 |
| **系统配置** | 系统参数 | 考勤时间、审批流程类型 |
| **业务字典** | 业务选项 | 请假类型、合同类型、职位级别 |

### 数据库设计建议

```sql
-- 字典分类表
CREATE TABLE dict_categories (
    id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
    code VARCHAR(50) UNIQUE NOT NULL,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    sort_order INT DEFAULT 0,
    created_at TIMESTAMPTZ DEFAULT CURRENT_TIMESTAMP
);

-- 字典项表
CREATE TABLE dict_items (
    id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
    category_code VARCHAR(50) NOT NULL REFERENCES dict_categories(code),
    code VARCHAR(50) NOT NULL,
    name VARCHAR(100) NOT NULL,
    value VARCHAR(255),
    sort_order INT DEFAULT 0,
    is_active BOOLEAN DEFAULT TRUE,
    created_at TIMESTAMPTZ DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(category_code, code)
);
```
