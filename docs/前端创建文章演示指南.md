# 文章发布完整流程演示指南

## 📋 第一步：初始化示例数据

运行数据库迁移添加示例数据：

```bash
cd /Users/xingzhai/POMP/backend/core
# 确保数据库连接正常后，SQLx会自动运行新的迁移
# 或者手动执行迁移
```

## 🚀 第二步：前端操作演示

### 1️⃣ 登录系统
1. 访问 `http://localhost:5173`
2. 使用管理员账号登录

### 2️⃣ 查看内容管理页面
1. 在左侧导航栏点击 **"内容管理"**
2. 你将看到：
   - 已发布的文章（绿色状态）
   - 草稿文章
   - 搜索和筛选功能

### 3️⃣ 创建新文章流程

#### 步骤 A：创建草稿
1. 点击右上角 **"新建文章"** 按钮
2. 填写文章信息：
   ```
   分类：产品展示
   标题：新一代工业物联网平台
   摘要：基于边缘计算的工业物联网解决方案
   内容：（你的文章内容）
   封面图片：https://images.unsplash.com/photo-1518770660439-4636190af475?w=800&h=600&fit=crop
   ```
3. 点击 **"保存草稿"**

#### 步骤 B：提交审核
1. 在文章列表中找到刚才创建的文章
2. 点击 **"提交审核"** 按钮
3. 文章状态变为 **"待审核"**

#### 步骤 C：审核通过
1. 使用管理员账号（如果不是当前账号）
2. 在待审核文章上点击 **"审核"**
3. 选择 **"通过"**
4. 添加审核意见
5. 点击提交

### 4️⃣ 发布到外网

#### 进入网站管理
1. 在左侧导航栏点击 **"网站管理"**

#### 配置网站设置
1. 在 **"网站设置"** 标签页：
   - 网站名称：三楷深发科技
   - 网站描述：专业的科技服务提供商
   - 主色调：蓝色
   - 联系方式：填写你的信息

#### 预览和发布
1. 点击 **"预览"** 查看网站效果
2. 点击 **"生成网站"** 生成静态文件
3. 点击 **"部署到公网"** 发布到 Cloudflare Pages

---

## 📊 文章状态流转

```
草稿 → 待审核 → (通过/拒绝) → 已发布 → 生成网站 → 部署公网
              ↓
          需要修改 → (重新编辑)
```

## 💡 创建文章建议

1. **内容结构**：
   - 清晰的标题
   - 简洁的摘要（200字以内）
   - 详细的正文内容
   - 吸引人的封面图

2. **图片使用**：
   - 使用 Unsplash、Pexels 等免费图库
   - 图片宽度建议 800px 以上
   - 保持图片比例 4:3 或 16:9

3. **发布流程**：
   - 先创建草稿
   - 编辑完善
   - 提交审核
   - 审核通过后自动发布
   - 在网站管理中部署到外网
