# 📋 职位智能填写指南

## 🏢 董事长 - 完整填写示例

### 快速填写步骤

## 方法一：使用模板（推荐）**

1. 进入 **组织架构管理** → **职位体系**
2. 点击顶部的 **"+董事长"** 模板按钮
3. 表单会自动填写，点击保存即可

## 方法二：手动填写**

| 字段 | 填写内容 | 说明 |
|------|---------|------|
| **职位名称** | 董事长 | 公司最高领导者职位 |
| **职位代码** | `chairman` | 建议使用英文小写，方便系统识别 |
| **职位级别** | 最高级（通常是 10 级） | 在职位级别中选择或创建高管级别 |
| **是否主管** | ✅ 是 | 董事长是最高主管 |
| **职位描述** | 公司最高决策者，负责制定公司战略方向、重大决策和董事会管理 | 自动生成的标准描述 |

## 🎯 填写规范建议

### 职位代码规范

- 使用小写字母
- 单词间用下划线分隔
- 保持唯一性
- 建议格式：`{部门}_{职位}`，如 `sales_manager`

### 职位级别设置建议

| 级别 | 职位示例 | 说明 |
|------|---------|------|
| 10 | 董事长、CEO | 最高决策者 |
| 9 | 总监、副总 | 高级管理层 |
| 7-8 | 经理、主管 | 中级管理层 |
| 4-6 | 专员、工程师 | 专业技术岗 |
| 1-3 | 助理、操作工 | 基层岗位 |

### 职位描述撰写技巧

✅ **职责描述公式**：`负责 + 工作领域 + 核心任务 + 目标`

**好的例子：**

- "负责公司财务管理、资金运作、财务报表和风险控制"
- "负责产品战略规划、团队管理和项目交付"

**避免：**

- "负责一些工作"（太笼统）
- "打杂"（太口语化）

## 💡 常见问题

**Q: 职位代码可以改吗？**  
A: 创建后建议不要修改，因为可能与其他系统关联。

**Q: 职位级别如何确定？**  
A: 参考"填写规范建议"中的级别表，也可在职位级别管理中自定义。

**Q: 可以创建多个董事长吗？**  
A: 技术上可以，但建议一个公司只设一个。
